Control de persianas=====================

El objetivo de nuestro proyecto es controlar las persianas, que se abren y se cierran de forma automática o remotamente por el usuario.
¿Qué es?
Es un sistema que mediante sensores detecta la luz solar y la lluvia, lo que provocará que las persianas se abran o se cierren según nuestras necesidades.
Si el sensor detecta la luz solar, se abrirán las persianas y apagarán las luces del interior y si no la detecta se cerrarán las persianas y se encenderán las luces del interior. Si detecta lluvia, las persianas se cerrarán.
